# 📚 Books Market Analysis using Web Scraping

## 📌 Project Overview

This project analyzes an online books dataset collected through **web scraping from the website *books.toscrape.com***.
The aim is to perform **data cleaning, exploratory data analysis (EDA), and visualization** to understand patterns in the online book market.

Dataset Details:

* Total Rows: **500**
* Total Columns: **10**

---

# 🎯 Project Objective

The objective of this project is to analyze the **books market dataset** and identify important patterns related to:

* Book Prices
* Ratings
* Categories
* Availability
* Reviews

Through this analysis we try to understand **how book ratings, categories, and prices are related.**

# 🔎 Exploratory Data Analysis (EDA)

The following analysis was performed:

### 1️⃣ Univariate Analysis

* Price Distribution
* Rating Distribution
* Category Distribution

### 2️⃣ Bivariate Analysis

* Price vs Rating
* Category vs Price
* Rating vs Availability

### 3️⃣ Multivariate Analysis

* Price vs Rating by Availability
* Correlation Heatmap
* Pairplot Analysis

---

# 📊 Key Insights

* Most books fall within a **moderate price range**.
* The majority of books have **ratings between 3 and 5**.
* Some categories contain **higher priced books**.
* Books with higher ratings sometimes have **slightly higher prices**.
* The dataset shows that **category and rating influence book pricing patterns**.
